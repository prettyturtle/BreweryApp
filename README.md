# 브루어리🍺

## (PunkAPI를 이용한 맥주리스트 앱)

- UIKit(Storyboard X)
- SnapKit
- Kingfisher

### 기능🍗

- PunkAPI를 사용하여 맥주리스트를 만든다
    - `UITableView`에서 커스텀 셀을 만들어 나타낸다
    - 커스텀 셀을 만들어 맥주의 간단한 정보를 표시한다
    - 무한 스크롤 구현
- 맥주 리스트에서 셀을 선택했을 때, 상세 정보를 표시하는 ViewController를 push한다
    - 상세정보 VC는 스크롤뷰로 구성되어있다
